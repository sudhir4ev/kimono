# @evooq/wetkit

Add dependency in `package.json`
```json
{
  "devDependencies": {
    "@evooq/wetkit": "*"
  }
}
```

Available cli commands

### `wetkit start`
Start dev server.


### `wetkit build`
Build production package


### `wetkit medusa build`
Build production package

### `medusa preview`
Preview medusa build

### `medusa setup`
Bootstrap a medusa microfrontend app.
- adds new entry file for medusa startup.
